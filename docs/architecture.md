# Nova 架构设计

## 当前状态

Nova 目前使用 Bevy + Rapier3D，但遇到物理不稳定问题：
- 机器人会倒地或散架
- 即使固定躯干，关节控制也不够精确
- 不适合专业机器人仿真

## 推荐方案：Rust + Isaac Lab

```
┌─────────────┐
│  Becoming   │ (决策系统)
└──────┬──────┘
       │
┌──────▼──────┐
│    Nova     │ (Rust 接口层)
│  Interface  │ - BrainInterface trait
└──────┬──────┘ - 数据验证和处理
       │ gRPC/ZMQ
┌──────▼──────┐
│ Isaac Lab   │ (Python 仿真环境)
│  Simulator  │ - GPU 加速物理
└─────────────┘ - 稳定的机器人仿真
```

## 为什么选择 Isaac Lab？

1. **NVIDIA 官方支持** - 最稳定，持续更新
2. **GPU 加速** - 适合大规模并行训练  
3. **专业机器人仿真** - 专门为机器人设计
4. **丰富的预置模型** - 包括人形机器人
5. **Python 生态** - 易于集成 ML 框架
